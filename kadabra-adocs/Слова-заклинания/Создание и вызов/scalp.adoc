=== scalp

Создаёт голову игрока.

==== Стоимость
2000 е.э.

==== Синтаксис
`"_имя игрока_" scalp`

==== Используемые цели
Блок, блок-цель, игрок.

==== Мощность
Влияет на число создаваемых голов.

==== Пример
`am "_имя игрока_" scalp`

==== Дополнительные сведения
Может создавать даже головы тех игроков, которые никогда не посещали сервер.
