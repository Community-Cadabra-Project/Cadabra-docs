=== primus

Устанавливает первый угол региона.

==== Стоимость
0 е.э.

==== Синтаксис
`primus regionus`

==== Используемые цели
Блок, блок-воздух.

==== Особые требования
Сначала нужно произнести `fixiro regionus`.

==== Мощность
Определяет высоту точки над выбранной целью.

==== Пример
`am unu! primus regionus`
