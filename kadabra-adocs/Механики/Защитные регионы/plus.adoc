=== plus

Добавляет игрока в список допущеных лиц региона.

==== Стоимость
0 е.э.

==== Синтаксис
`_"имя_игрока"_ plus`

==== Используемые цели
Регион, игрок.

==== Особые требования
Игрок должен быть онлайн.

==== Мощность
Выбирает регион.

==== Пример
`am _"имя_игрока"_ plus regionus`
