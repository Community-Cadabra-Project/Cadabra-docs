=== telepata

Накладывает эффект, передающий все слова в мысли от заклинателю к игроку-цели.

==== Стоимость
`_мощность_`*40е.э.

==== Синтаксис
`telepata`

==== Используемые цели
Игрок.

==== Особые требования
Игрок должен быть онлайн.

==== Мощность
Мощность определяет время действия эффекта телепатии.

==== Пример
`am personi "_имя_цели_" telepata`
